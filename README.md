# Анализ настроения текстов на русском языке с помощью RuBERT

Данный проект представляет собой модуль для автоматического определения тональности текстов на русском языке. В качестве модели используется предобученный **RuBERT (rubert-tiny-sentiment-balanced)** из библиотеки `transformers`.

## Задача
Преобразовать текстовые записи (например, дневниковые заметки, посты в соцсетях) в числовую оценку настроения от 0 (крайне негативное) до 1 (крайне позитивное) и провести первичный анализ.

## Используемые технологии
- Python 3.11
- transformers (Hugging Face)
- PyTorch
- pandas, matplotlib, seaborn

## Как это работает
1. Модель `cointegrated/rubert-tiny-sentiment-balanced` классифицирует текст на три категории: `positive`, `negative`, `neutral`.
2. Полученная метка и confidence‑score преобразуются в единую числовую оценку:
   - positive → 0.6 + 0.4 * score
   - neutral → 0.4 + 0.2 * score
   - negative → 0.4 * score
3. Для демонстрации сгенерирован датасет из 30 текстовых записей за месяц. Модель проинференсирована для каждой записи.
4. Построены визуализации:
   - динамика настроения по дням
   - распределение эмоций (радость/нейтрально/грусть)
   - статистика (среднее, максимум, минимум)
   - тренд со скользящим средним (3 и 7 дней)

## Результаты
- Среднее настроение: 0.62
- Выявлен положительный тренд: изменение средней оценки с 0.60 в первую неделю до 0.69 в последнюю.
- Распределение дней: 30% радостных, 36.7% нейтральных, 33.3% грустных.

Графики можно посмотреть в ноутбуке.
